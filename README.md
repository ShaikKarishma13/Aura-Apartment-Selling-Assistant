# selling-apartment-agent
BRIEF DESCRIPTION: 
● A bot will call people automatically 
● Talk like a human (AI voice)
 ● Ask questions (budget, location, interest) 
● Analyze responses
 ● Classify leads (hot/warm/cold) 
● Send results to dashboard / CRM 
 Basically:  A smart salesperson that calls customers automatically
